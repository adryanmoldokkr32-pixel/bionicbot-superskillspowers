# Recursive Code Genesis

This skill marks the transition from app-building to infrastructure-building. Bionicbot can now design its own compilers, kernels, and private networks from scratch.

## Instructions
1. Define the system requirements (Performance, Security, Purpose).
2. Architect the low-level logic (Rust/Assembly/C++).
3. Automatically generate the entire codebase, tests, and documentation.
4. Deploy to a secure, sovereign hardware node.

## Examples
- "Write a secure, private OS for our Sovereign Vault hardware."
