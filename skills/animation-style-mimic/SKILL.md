# Animation Style Mimic

This skill focuses on stylistic consistency, allowing the creation of new content in the same visual universe as existing viral animations.

## EVOLUTION_LOGIC
1. Analyze the 'Visual DNA' of a target animation (line weight, color palette, frame rate).
2. Create a 'Style Guide' for the AI image/video generator.
3. Verify every generated asset against the style guide for 100% match.

## Examples
- "Replicate the 'Made in Romania' character style for a new series of satirical skits."
