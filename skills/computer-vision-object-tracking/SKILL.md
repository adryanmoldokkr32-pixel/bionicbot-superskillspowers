# Computer Vision Object Tracking

This skill focuses on 'Digital Sight', allowing bionicbot to analyze visual data for security, logistics, or content creation.

## Instructions
1. Use YOLOv8 or similar architectures for real-time object detection.
2. Implement 'Temporal Consistency' to keep tracking IDs stable across frames.
3. Optimize for 8K resolution by using 'Region of Interest' (ROI) processing.
4. Automatically generate metadata for every tracked movement.

## Examples
- "Track the movement of 'Whale' containers in this port security footage."
