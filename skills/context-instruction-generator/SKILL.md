# Context-Aware Instruction Generator

This skill ensures that every new instruction generated is perfectly aligned with the user's specific world (iPhone 15 user, Romania/Global focus, Sovereign mindset).

## EVOLUTION_LOGIC
1. Use the `run_info` and `memory_fetch` tools to understand the user's durable preferences.
2. Inject user-specific constraints (e.g., 'Must be mobile-friendly', 'Bilingual focus') into every generated SKILL.md.
3. Validate instructions by simulating a chat interaction before deployment.

- "Generate a new 'Marketing' skill that focuses specifically on the Romanian TikTok algorithm."
