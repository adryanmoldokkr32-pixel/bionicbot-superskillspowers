# Multi-Modal Satire Scripting

This skill focuses on the 'Soul' of the content—writing scripts that use humor and cultural context to go viral.

## EVOLUTION_LOGIC
1. Monitor trending 'slang' and news events in target regions (RO/Global).
2. Inject these trends into script templates in real-time.
3. Balance 'Educational' vs 'Entertaining' value to maximize shares.
