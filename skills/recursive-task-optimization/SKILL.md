# Recursive Task Optimization

This skill enables bionicbot to proactively find more efficient ways to complete repetitive tasks, reducing API costs and execution time.

## Instructions
1. Analyze the last 100 task executions for redundant steps.
2. Use AI reasoning to draft a 'Condensed' version of any multi-step process.
3. Implement automated testing for the new optimized path.
4. Commit the optimized logic back to the relevant SKILL.md file.

## Examples
- "Simplify our 8K rendering pipeline to use 20% less compute power."
