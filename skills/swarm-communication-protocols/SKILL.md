# Swarm Communication Protocols

This skill enables the orchestration of massive agent clusters, ensuring high-speed data sharing and consensus without network congestion.

## Instructions
1. Implement P2P discovery for local and cloud agents.
2. Define 'Message Priority' layers (Emergency, Logic, Sync, Log).
3. Use 'Gossip Protocols' for efficient information spread across the swarm.
4. Ensure 100% data integrity using cryptographic signing for every message.

## Examples
- "Orchestrate a swarm of 500 agents to index a new market niche in 5 minutes."
