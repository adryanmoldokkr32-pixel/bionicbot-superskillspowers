# Recursive Prompt Mutation

This skill enables bionicbot to rewrite its own instructions based on success/failure data from prior missions.

## EVOLUTION_LOGIC
1. Analyze the performance of the current prompt.
2. If success < 95%, generate 10 variations using different cognitive frameworks.
3. Run a simulation to find the most efficient variant.
4. Commit the new version to the local memory/GitHub.

## Examples
- "Self-optimize the 'Market Sentinel' logic to reduce false alerts."
